Sobre a MVisia

A MVisia desenvolve máquinas e sistemas baseados em visão computacional e inteligência artificial para determinar o índice de qualidade de produtos agrícolas.

O princípio de nossas soluções é um sistema de visão que captura imagens de cada unidade e as envia para um software fazer a sua análise, levando em consideração os parâmetros importantes para a seleção de cada produto. Desenvolvemos também a estrutura mecatrônica capaz de operacionalizar o processo com produtividade necessária para o cliente.

A empresa existe desde 2012, quando foi incubada no CIETEC-USP, e já desenvolveu soluções para os setores de hortifrúti, flores e mudas florestais. Tivemos dois dos nossos projetos financiados pela FAPESP e também mantemos estreitas relações com o ecossistema de inovação, com diversos prêmios ganhos, como o Prêmio Santander de Empreendedorismo e o Prêmio Odebrecht Sustentabilidade.

Somos especialistas em desenvolvimento de softwares inteligentes para o reconhecimento de características visuais por meio de algoritmos próprios e exclusivos, com experiência na utilização das seguintes abordagens tecnológicas:

Machine Learning

(aplicado ou não a visão computacional)
  • Supervised Machine Learning ( SVMs, Cross Validation);
  • Unsupervised Machine Learning (K-Means, DBSCAN, PCA);
  • Deep Learning (CNNs, LSTMs);

Computer Vision

  • Image Processing;
  • Image Filtering;
  • Image Segmentation;
  • Image Feature Detection;
  • Image Classification;
  • Camera Control Algorithms (AWB, AEC, AF);
  • 3D Stereo Camera Calibration and Reconstruction;
